# Knowledge Agents - Enterprise AI Platform Website

## Overview
A complete, modern, enterprise-grade website for Knowledge Agents - an AI-powered platform for intelligent document analysis, validation, and automation.

## Design Philosophy
- **Refined Editorial Minimalism**: Clean, spacious layouts inspired by Stripe, Notion, and OpenAI
- **Professional Typography**: Fraunces for display headings, Inter for body text, Berkeley Mono for code
- **Color Palette**: Deep blues (#0A2540), teals (#14B8A6), and cyan accents (#00D4FF) for a professional, trustworthy feel
- **Subtle Animations**: Polished micro-interactions without gimmicks
- **Enterprise-Grade**: Credible, premium design suitable for Fortune 500 companies

## Customization Guide

### Colors
Edit CSS variables in `styles.css`:
```css
--primary: #0A2540;        /* Main brand color */
--accent: #00D4FF;         /* Accent/CTA color */
--teal: #14B8A6;          /* Secondary accent */
```

### Typography
Font families can be changed in CSS:
```css
--font-display: 'Fraunces', 'Georgia', serif;
--font-body: 'Inter', sans-serif;
```

### Content
All content is easily editable in HTML files. Search for class names:
- `.hero-title` - Main headlines
- `.hero-subtitle` - Subheadlines
- `.section-title` - Section headers
- `.feature-description` - Feature descriptions
