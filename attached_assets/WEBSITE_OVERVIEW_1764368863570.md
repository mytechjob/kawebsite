## üîß Customization Guide

### Change Colors
Edit in `styles.css` (lines 2-16):
```css
--primary: #0A2540;
--accent: #00D4FF;
--teal: #14B8A6;
```

### Change Fonts
Edit in `styles.css` (lines 18-20):
```css
--font-display: 'Your Display Font', serif;
--font-body: 'Your Body Font', sans-serif;
```

### Modify CTAs
Global CTAs to update:
- "Request a Demo"
- "Talk to Sales"
- "Try Knowledge Agents"

Search and replace across all HTML files.

## üÜò Common Questions

**Q: How do I add more pages?**
A: Copy an existing HTML file, update the content, add link to navigation in all pages.

**Q: How do I change the logo?**
A: Replace the SVG in the `.nav-logo` section of each page, or use an `<img>` tag instead.

**Q: Can I use a different font?**
A: Yes! Just update the CSS variables and include the font from Google Fonts or your hosting.

**Q: How do I make the forms work?**
A: You'll need a backend service like:
- Formspree (easiest)
- Netlify Forms (if hosted on Netlify)
- Custom backend (PHP, Node.js, etc.)
- Email integration service
